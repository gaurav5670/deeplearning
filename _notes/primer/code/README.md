# Code Scaffold


1. [RNNLIB](https://github.com/szcom/rnnlib) is a recurrent neural network library for sequence learning problems. Forked from Alex Graves work 
2. A [deep Q learning](https://github.com/nivwusquorum/tensorflow-deepq) demonstration using Google Tensorflow 
3. [CharRNN](https://github.com/karpathy/char-rnn) : Multi-layer Recurrent Neural Networks (LSTM, GRU, RNN) for character-level language models in Torch 
4. [NeuralTalk](https://github.com/karpathy/neuraltalk) is a Python+numpy project for learning Multimodal Recurrent Neural Networks that describe images with sentences
